# userlogin
前后端分离用户登录功能

# 环境
    linux
    python3         (flask)
    nodejs          (vue3)

# 目录结构
    ├── README.md
    ├── backend             # python3项目
    ├── frontend            # 用户登录页面
    ├── web1                # web1项目
    └── env.sh              # 运行项目脚本

# 说明
##### 下载代码

    git clone https://github.com/qiesuijifengqifei/userlogin.git

##### 导入项目脚本
    source env.sh

    # 子命令含义
    
    init_env            # 安装项目调试运行的依赖
    run_all             # 运行项目 backend , frontend , web1
    stop_run            # 停止运行的所有项目

# 访问
    http://127.0.0.1:8082/web1/home
